## Bem-vindo(a)! 👋

Este repositório reúne as **resoluções das fichas práticas**, exemplos e materiais de apoio desenvolvidos no âmbito da turma **Dev Stream 06/26** das **Academias Natixis**.

O principal objetivo é apoiar os formandos na consolidação dos conteúdos abordados ao longo da formação, com foco no desenvolvimento backend utilizando **Java** e **Spring Boot**, promovendo a prática, o raciocínio lógico, a autonomia e a aplicação de boas práticas de desenvolvimento de software.

---

## 📚 Conteúdos Abordados

Ao longo deste repositório poderão ser encontrados exercícios, exemplos e resoluções relacionados com os seguintes temas:

* **Fundamentos de Java aplicados ao desenvolvimento backend**
* **Introdução ao Spring Boot**
* **Criação de APIs REST**
* **Controllers, Services e Repositories**
* **Injeção de Dependências**
* **Validação de Dados**
* **Tratamento de Erros**
* **Persistência de Dados**
* **Spring Data JPA**
* **Relacionamentos entre Entidades**
* **DTOs e Mapeamento de Dados**
* **Boas Práticas na Organização de Projetos Spring Boot**
* **Testes e Validação de Funcionalidades**
